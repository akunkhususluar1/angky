# Portfolio — Angky

This repository contains a simple personal portfolio site built with plain HTML and CSS. It is intended to be published with GitHub Pages.

To publish:
1. Add these files to the repository root: `index.html`, `styles.css`.
2. Enable GitHub Pages in repository Settings → Pages and select the `main` branch (root).
3. Your site will be available at: `https://akunkhususluar1.github.io/angky/`

Customize this site by editing the HTML and CSS files.
